# Loopback

Loopback is an outbound to send traffic back to the router.

This is often used together with chained proxy support.

# Loopback Outbound

  • Name: loopback
  • Type: Outbound Protocol
  • ID: outbound.loopback

inboundTag: string

The inbound tag for loopback traffic.
